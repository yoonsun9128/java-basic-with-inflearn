* 기본형 VS 참조형
기본형 : int, long, 처럼 변수에 사용할 값을 직접 넣울 수 있는 타입
-> 실제 사용하는 값을 변수에 담을 수 있음
-> 실제 사용하는 값으로 인해서 직접 계산할 수 있다
-> 변수를 선언할때 모두 소문자로 시작한다.
-> java가 기본적으로 제공하는 타입
참조형 : 직접사용하는 값이 아닌, 위치(참조, 주소) 참조값이 들어가 있음
-> 객체와 배열이 있음
-> 참조값으로 인해 직접 계산할 수 없음 but .을 사용해서 객체의 기본 변수로 접근할 경우 계산이 가능해

** 참고 String == class
=> 참조형 이지만 문자형은 매우 자주 다뤄지기 때문에 java에서 편의 기능으로 제공하는것

** 대원칙 : java는 항상 변수의 값을 복사해서 대입한다.
